Windows:

    Installing Anaconda:

        1. Go to https://www.anaconda.com/ 
        2. On Menu at the top hover over Products, and select Individual Edition
        3. Click Download, then Click "64-Bit Graphical Installer" (or 32-bit if you are on 32-bit machine)
        4. When done downloading, begin installation 
        5. Be sure NOT to add conda to path (this will be an option in the installer)
    
    Setting up New Conda Environment:

        1. Type "Anaconda" into search bar
        2. Open "Anaconda Powershell Prompt" (pin to taskbar for future)
        3. In Powershell
